# Customer Trends Data Analysis

## Project Overview
This project analyzes retail customer shopping behavior to uncover trends that improve revenue performance, customer engagement, and product strategy.

The analysis includes data cleaning, database integration, SQL insights, and interactive dashboards.

---

## Tools & Technologies
- Python (Pandas, SQLAlchemy)
- PostgreSQL
- SQL
- Power BI

---

## Project Workflow
1. Data Cleaning & Feature Engineering (Python)
2. Database Integration (PostgreSQL)
3. SQL-Based Analysis
4. Dashboard Development (Power BI)
5. Business Insights & Recommendations

---

## Business Insights
- Identified top-performing product categories
- Observed strong seasonal trends in revenue
- Analyzed demographic purchasing behavior
- Evaluated impact of discounts on sales performance

- customer_shopping_behavior.ipynb → Data preparation & database integration
